# Tokyo Table Flow をスマホで渡すメモ

このアプリはスマホ対応のWebアプリです。`index.html` を開くだけでも使えますが、友達に渡すならWebに置いてURLで渡すのが一番自然です。

## 明日渡すなら

1. このフォルダを GitHub Pages、Netlify、Vercel などにアップロードします。
2. 発行されたURLをスマホで開きます。
3. iPhoneなら共有ボタンから「ホーム画面に追加」、Androidならブラウザメニューから「アプリをインストール」または「ホーム画面に追加」を選びます。

## ローカル確認

このフォルダで簡易サーバーを起動すると確認できます。

```powershell
py -3.13 -m http.server 8765 --bind 127.0.0.1
```

その後、ブラウザで `http://127.0.0.1:8765/index.html` を開きます。

## 注意

現在の混雑状況はサンプル推定データです。実店舗のリアルタイム混雑を出すには、店舗データ、予約台帳、Google Places API などのデータ連携が必要です。
